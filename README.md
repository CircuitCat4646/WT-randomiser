# WT-randomiser

This is a simple randomiser I made for War Thunder.

There are 2 tabs, vehicle manager, and the randomiser itself.
In the vehicle manager you fill out the information about a vehicle and click add entry, this adds an entry to a vehicles.csv file
Once you have enough vehicles added to the csv you can use the vehicle randomiser, you can also use the added filters to narrow down your random pool.

The vehicles.csv file added is an example I created.

This is a passion project made over the course of about a day for fun, don't expect any major updates to it.
